🐍 Python - Basic Chat

from wizard import Wizard

client = Wizard(api_key="YOUR_API_KEY")

response = client.chat.create(
    messages=[
        {"role": "system", "content": "You are JARVIS, a helpful AI."},
        {"role": "user", "content": "What's the weather like today?"}
    ],
    model="wizard-jarvis",
    temperature=0.7
)

print(response.choices[0].message.content)

🐍 Python - Streaming Response

from wizard import Wizard

client = Wizard(api_key="YOUR_API_KEY")

stream = client.chat.create(
    messages=[{"role": "user", "content": "Tell me a story"}],
    model="wizard-fun",
    stream=True
)

for chunk in stream:
    print(chunk.choices[0].delta.content, end="")

📜 JavaScript - Using Fetch API

const response = await fetch('https://api.wizardai.dpdns.org/v1/chat/completions', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        model: 'wizard-jarvis',
        messages: [
            { role: 'user', content: 'Hello!' }
        ]
    })
});

const data = await response.json();
console.log(data.choices[0].message.content);

📡 cURL - Direct API Call

curl -X POST https://api.wizardai.dpdns.org/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wizard-jarvis",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'

💡 Pro Tips

  • Use wizard-fast for quick, simple responses
  • Use wizard-oracle for complex, thoughtful answers
  • Enable streaming for real-time responses in chat applications
  • Store your API key in environment variables, never hardcode it
  • Monitor your usage in the Analytics dashboard
